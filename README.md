# Python学生成绩管理系统（源码+开发文档）

## 核心功能
- 用户管理
- 班级管理
- 学生管理
- 成绩管理
- 数据总览
- 条件筛选
- 分页查询

## 适用场景
- 学校、教育机构
- 教师个人成绩管理
- 学科成绩分析

## 技术特点
- 后端：Python、Flask
- 前端：HTML、CSS、JavaScript、jQuery
- 数据库：MySQL
- 开发工具：Vscode、Navicat
- 支持环境：Python3.0以上、MySQL8.0
